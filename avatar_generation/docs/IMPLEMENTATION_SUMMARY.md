## 设计亮点

### 1. 参考成功经验
完全基于现有`diverse_prompt_generator.py`的成功架构：
- 使用相同的权重分布系统
- 相同的可重现种子机制（seed = 42 + i）
- 相同的代码结构和命名规范

### 2. 具体化描述
不使用模糊词汇，全部使用具体描述：
- ❌ "美食" → ✅ "juicy classic cheeseburger with melted cheddar"
- ❌ "游戏玩家" → ✅ "competitive esports player playing Valorant ranked match"

### 3. 真实生活感
每张图都包含关键词确保iPhone随拍风格：
- "Shot on iPhone"
- "casual lifestyle photo" / "casual gaming photo"
- "natural indoor lighting"
- "authentic moment"

### 4. 权重分布系统
确保合理的分布，避免极端场景过多：
```python
self.content_types = [
    {"type": "pure_food", "weight": 0.40},      # 常见场景权重高
    {"type": "person_eating", "weight": 0.35},
    {"type": "food_blogger", "weight": 0.25},   # 特殊场景权重低
]
```

### 5. 美国审美
- 所有人物反映美国族裔分布（白人60%、拉丁裔18%、非裔13%等）
- 美食都是美式常见食物
- 游戏都是美国流行的游戏
- 场景都是美国典型环境
